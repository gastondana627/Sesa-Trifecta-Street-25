Technical Roadmap

An overview of the project's infrastructure, integration strategy, and development workflow.

Infrastructure & Integration

  • Current Deployments: Astro Archive (Streamlit), Relic (Vercel), Peata (Streamlit).
  • Long-Term Vision: Unify all apps behind a central Next.js landing hub deployed on Vercel for scalability and security.
  • Future Goal: Consider containerization with Docker for improved portability, consistency, and simplified deployment across environments.

CI/CD Pipeline (Future)

  • Automation: Implement a `.github/workflows/deploy.yml` file.
  • Triggers: Automatically redeploy an application when changes are detected in its specific directory (`astro-archive/`, `relic/`, `peata/`).
  • Efficiency: Streamline the deployment process, ensuring rapid and reliable updates.
